Interpolates from a coarse patch to a fine patche.

Parameters
[in]mx,my,mzthe number cells in the x, y, and z directions, excluding ghost
[in]mbcthe number of ghost cells
[in]meqnthe number of equations
[in]qcoarsethe coarse solution
[out]qfinethe fine solution
[in]areacoarsethe area of the coarse cells
