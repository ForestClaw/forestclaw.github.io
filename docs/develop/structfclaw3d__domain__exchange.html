Memory where the ghost patch data actually lives.

◆ patch_data

void** fclaw3d_domain_exchange::patch_data

One pointer per processor-local exchange patch in order, for a total count of domain->num_exchange_patches.

This applies precisely to local patches that touch the parallel boundary from the inside, i.e., if (flags & FCLAW3D_PATCH_ON_PARALLEL_BOUNDARY).
