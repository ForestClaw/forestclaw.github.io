Destruct a dimension-specific domain and deinitialize patch data.

Parameters
[in]domainValid dimension-independent domain structure.
[in]deleThis callback is pointed to each existing fclaw_patch whose 3D data it is supposed to deinitialize, as well as the patch user data. Note, just don't allocate the patch itself! May be NULL for no callback.
[in,out]userPointer passed through to init.

Construct a dimension-specific domain and initialize patch data.

Parameters
[in]domainFreshly created, valid domain structure without any user data attached to it.
[in]initThis callback is pointed to a freshly allocated fclaw_patch whose 3D data it is supposed to fill with fitting values. This includes allocating and filling the patch user pointer, which the callback can do using the third parameter to this function, user. May be NULL for no callback.
[in,out]userPointer passed through to init.
