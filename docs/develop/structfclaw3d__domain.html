Detailed Description

The domain structure is a collection of blocks.

The domain structure gives a processor local view of the grid hierarchy. Unless explicitly noted otherwise, all variables are processor local, i.e., they are generally different on each processor. Variables that are synchronized and shared between processors are denoted global.
