Adds wave corrections at same level interfaces.


This accounts for metric mismatches that can occur at block boundaries.

Parameters
[in]mx,mythe number of cells in the x and y directions
[in]mbcthe number of ghost cells
[in]meqnthe number of ghost cells
[in]idirthe direction of the interface 0 for bottom/top 1 for left/right
[in]iface_thisthe interface on the this patch
[in]this_blocknothe block number of the coarse patch
[in]nbr_blocknothe block number of the fine patch
[in]area0,area1,area2,area3area of cells along the edges on this patch
[in,out]qthisthe solution on this patch
[in]fpthis0,fmthis1,gpthis2,gmthis3the fluxes along the edges of the this patch
[in]fmfine0,fmfine1,gpfine2,fpfine3the fluxes along the dges of the neighbor patch
[in]efthis0,efthis1,efthis2,efthis3the values of the flux function along the this patch edges
[in]efnbr0,efnbr1,efnbr2,efnbr3the values of the flux function along the neighbor patch edges
[in,out]qnbr_dummywork vector for determining values in the fine patch
[in]transform_cptrthe pointer to the fclaw2d_patch_transform_data struct
