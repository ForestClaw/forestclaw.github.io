typedef void(* fclaw_family_callback_t) (fclaw_global_t *global, fclaw_patch_t *family[], int blockno, int patchno, void *user)

Dimension-independent callback prototype for family iterators.

We iterate over local families of patches only.

Parameters
[in]domainDimension-independent global context.
[in]familyArray of the patches in the family.
[in]blocknoBlock number of processed patches.
[in]patchnoThe local number of the first sibling patch in the family, relative to the block. The remaining 3 (2D) or 7 (3D) are consecutive.
[in,out]userData that was passed into the iterator functions.
