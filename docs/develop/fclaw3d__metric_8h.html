)

Get the scalar metrics of a patch.

Parameters
[in]globglob the global context
[in]patchthe patch context
[out]areathe area of each cell in the patch
[out]faceareas the three faces in each cell. An array of dimension (-mbc:mx+mbc+2,-mbc:my+mbc+2,-mbc:my+mbc+2,3). (i,j,k,1) contains the area of face with normal (1,0,0) (left face) (i,j,k,2) contains the area of face with normal (0,1,0) (front face) (i,j,k,3) contains the area of face with normal (0,0,1) (bottom face)
