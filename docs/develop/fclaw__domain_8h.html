typedef void(* fclaw_domain_callback_t) (fclaw_domain_t *domain, fclaw_patch_t *patch, int blockno, int patchno, void *user)

Dimension-independent callback prototype for the patch iterators.

We iterate over local patches only.

Parameters
[in]domainDimension-independent domain structure.
[in]patchThe local patch currently processed by the iterator.
[in]blocknoBlock number of processed patch.
[in]patchnoPatch number within block of processed patch.
[in,out]userData that was passed into the iterator functions.

Destruct a dimension-specific domain and its patch data.

Parameters
[in,out]domainInitialized, valid domain structure. On output, the pointer is invalidated.
[in]deleThis callback is pointed to an existing fclaw_patch whose data it is supposed to delete. This includes deleting the patch user pointer, which the callback can do using the third parameter to this function, user. Just do not delete the patch itself!
[in,out]userPointer passed through to init.
