◆ skip_refined

int fclaw2d_domain_persist::skip_refined

Boolean: If true, the patch data of patches that were refined during the most recent call to fclaw2d_domain_adapt is only packed for the patches with child id 0 in fclaw2d_domain_iterate_pack.

The unpack callback in fclaw2d_domain_iterate_unpack will be invoked for all children of a recently refined patch. The provided data will be the data packed by the patch with child id 0. For this partitioning method to work as desired skip_refined has to be set before the most recent call to fclaw2d_domain_adapt.
