Detailed Description

Routines which accumulate fluxes, waves and add corrections to coarse grid at edges of both same size grids and coarse/fine interfaces.

  1. clawpack46_time_sync_store_flux (before step update)
  2. clawpack46_time_sync_accumulate_waves (after step update)
  3. clawpack46_fort_time_sync_f2c ()

This stores the value of the flux function at cell centers.

Called BEFORE step update. At k=0, the flux at the the first interior interior cell is stored; At k=1, the flux evaluated in the ghost cell is stored; If there is no flux function, we should set up a dummy function that returns zero.

These will be stored for each grid and used to compute corrections.

Parameters
[in]mx,mythe number of cells in the x and y directions
[in]mbcthe number of ghost cells
[in]meqnthe number of ghost cells
[in]mauxthe number of aux equations
[in]blocknothe block number
[in]patchnothe patch number
[in]dtthe time step size
[in]el0,el1,el2,el3the edge lengths along the edges
[in]qthe solution
[in]auxaux array
[in,out]flux0,flux1,flux2,flux3the values of the flux function
